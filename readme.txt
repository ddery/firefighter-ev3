Cara mengeksekusi program : 

1. Buka aplikasi ROBOTC
2. Buka file source dalam ROBOTC
3. Pastikan bahwa file level sudah berada di dalam direktori ROBOTC
4. Tekan F5 (untuk compile sekaligus download)
5. Masuk ke RVW dengan file level yang diinginkan
6. Tekan tombol eksekusi program di pojok kiri bawah layar RVW
7. Selamat menyaksikan !
